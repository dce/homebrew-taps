# dce/homebrew-taps

Homebrew tap for personal CLI tools published by [@dce](https://github.com/dce).

## Available formulae

- [`mdrenum`](https://github.com/dce/mdrenum): Keep numbered reference-style Markdown links in sequential order.
- [`obrol`](https://github.com/dce/obrol): Obsidian Rolodex CLI for managing contacts stored as Markdown.

## Install the tap

```sh
brew tap dce/taps
```

## Install a tool

```sh
brew install mdrenum
```

## Upgrade

```sh
brew update
brew upgrade mdrenum
```

## Uninstall

```sh
brew uninstall mdrenum
```
